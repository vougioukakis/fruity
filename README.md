<h3>Fruity</h3>

A web app with a simple fruit image recognition model (neural network) that provides nutritional information about the recognized fruit, as well as random fruit facts!
The application calls my own API to retrieve the information and the facts. The API also supports adding
new fruit facts from the website.

The server is made using Quart with Python, and the API using Express with JavaScript.

This is a prototype design of the website, which is work in progress.
